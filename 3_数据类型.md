## 数据类型(简单介绍)

### 数值类型 int及float

整数类型和浮点数类型

整数 int

浮点数 float



### 布尔值类型 bool

True (真、对)

False(假、错)

用于判断



### 字符串类型 str

使用 ' ' 所框住的就是字符串



### 列表类型 list

使用中括号 [ ] 框住的，可放数值、字符串、变量名，使用逗号隔开,



### 元组类型 tuple

使用括号( )框住的，与列表相似



### 集合类型 set

使用大括号{ }框住的，特点为无序、不重复
若设置时有重复值，输出就会忽略掉



### 字典类型 dict

键值对 (key:value)

使用大括号{ }框住的, 与集合不同的是内容须为「键值对」



### 查看数据类型的方法： type()
