# 认识Python

### Python简介

- Python是由荷兰人吉多·范罗苏姆（Guido von Rossum）(简称龟叔)发明的一种编程语言，是目前世界上最受欢迎和拥有最多用户群体的编程语言。
- Python是一门解释形语言
- Python的优点：
  - 简单明瞭，易上手
  - 开发效率高
  - 第三方库众多
  - 在各种平台均能使用
- Python的缺点
  - 执行速度较编译形语言慢，但一般人的感知不强
